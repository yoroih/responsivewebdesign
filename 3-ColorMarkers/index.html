CSS Color Markers

RGB Colors

RGB refers to the three primary colors of light used to create a wide range of colors. Each color is represented by a numerical value ranging from 0 to 255 for each component of red, green, and blue.

The general format for defining an RGB color in CSS is as follows:

      selector {
        color: rgb(red-value, green-value, blue-value);
      }
